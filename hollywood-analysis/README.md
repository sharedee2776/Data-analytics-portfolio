#  Hollywood Movie Analysis (Tableau)

###  Objective
Explore Hollywood movie performance based on budget, revenue, ratings, and studio output.

###  Dashboard Highlights
- Budget vs. revenue scatterplot  
- Rating distribution  
- Top studios by profitability  
- Yearly trend charts  

###  Insights
- Certain studios consistently outperform competitors.  
- Budgets do not always correlate strongly with revenue.  
- Mid-budget films often deliver strong ROI.
